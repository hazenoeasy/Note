# Spring 简介
## Spring 是什么
  Spring是分层的JAVA SE/EE框架 以IoC AoP为内核
## Spring 优势
 - 方便解藕  将对象间的依赖关系交给Spring控制
 - AOP 支持
 - 声明式事务的支持
 - 降低JavaEE API的使用难度
## Spring 体系结构
 - Core Container: Beans Core Context SpEL
 - AOP Aspects 
 - Data Access: JDBC  
 - Web: Servlet WebSocket

# Spring 核心机制
- IoC / DI IOC产生各种对象 DI进行关联。
