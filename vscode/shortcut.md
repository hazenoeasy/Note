

快捷键
Command + Shift + F  搜索文件夹下所有文件

Command + Shift + T  重新打开关闭的页面

command + T  创建新的terminal

command +option + l r  切换tab

command + option + up down 选择多行操作

Control + Option + R 重启界面

command +shift + N 创建新的界面

control + ` 打开关闭terminal

fn+ 箭头  行操作

option + delete 删除单词

command + x 删除整行
